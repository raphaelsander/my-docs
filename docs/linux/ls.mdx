# LS

Liste informações dos arquivos (no diretório atual por padrão).

---

Listar pastas e arquivos do diretório:

```bash
ls
```

Listar pastas e arquivos, inclusive os ocultos (que iniciam com ``.`` no nome) 
do diretório:

```bash
ls -a
```

Listar pastas e arquivos, inclusive os ocultos do diretório, em formato de 
lista:

```bash
ls -la
```
