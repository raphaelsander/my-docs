title: ls
---

Liste arquivos, diretórios e links simbólicos.

---

Listar pastas e arquivos do diretório:

```bash
ls
```

Listar pastas e arquivos, inclusive os ocultos (que iniciam com `.` no nome) do
diretório:

```bash
ls -a
```

Listar pastas e arquivos, inclusive os ocultos do diretório, em formato de
lista:

```bash
ls -la
```

Listar apenas diretórios:

```bash
ls ./*/ -d
```

```bash
ls path/*/ -d
```

:::warning
Esse comando não exibirá diretórios ocultos (que iniciam com `.` no nome).
:::
