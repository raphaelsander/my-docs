Whois é um protocolo de consulta utilizado para obter informações sobre a
propriedade e os detalhes de registro de um domínio ou endereço IP.

---

## Cliente Whois

A maioria das distribuições Linux inclui um cliente whois por padrão.

## Utilização do Comando Whois

Para realizar uma consulta whois para um domínio específico, utilize o seguinte
comando:

```bash
whois site.com.br
```

## Consulta em um Servidor Específico

Para consultar informações sobre um domínio em um servidor específico, utilize a
opção `-h` para especificar o servidor:

```bash
whois desec.com.br -h whois.registro.br
```

:::info
É necessário que a autoridade do servidor especificado tenha jurisdição sobre o
domínio consultado. Por exemplo, não é possível realizar uma pesquisa whois de
um domínio brasileiro no servidor de um órgão no Japão.
:::
