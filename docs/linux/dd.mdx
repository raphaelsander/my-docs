O `dd` é um programa de clonagem que mantem as propriedades das partições
clonadas, inclusive bits nulos são clonados.

---

## Clonando Flash Drive para Arquivo

Insira o cartão SD ou pendrive e verifique o nome do dispositivo (geralmente
mmcblkX ou sdcX):

```bash
sudo fdisk -l
```

No meu caso, o cartão SD é `/dev/mmsbkl0`. Agora você tem que desmontar o
dispositivo:

```bash
sudo umount /dev/mmcblk0
```

Agora, para criar uma imagem do dispositivo:

```bash
sudo dd if=/dev/mmcblk0 of=~/sd-card-copy.img
```

:::info
Isso vai demorar um pouco dependendo do tamanho do armazenamento clonado.
:::

## Escrevendo Imagem no Flash Drive

Escrevendo a imagem para um dispositivo:

```bash
sudo dd if=~/sd-card-copy.img of=/dev/mmcblk0
```

:::info
Normalmente a operação de gravação é muito mais lenta que de leitura.
:::
